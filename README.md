Weather App 2.0 - React Project
A responsive React weather application providing real-time weather data and 5-day forecasts with temperature unit conversion.

Tech Stack:
React 18.3.1 with hooks
Bootstrap 5.3.3 for responsive design
Axios for API calls
OpenWeatherMap & SheCodes APIs
Animated weather icons

Key Features:
City search with live weather data
Current conditions (temperature, humidity, wind, feels-like)
5-day forecast with daily highs/lows
Celsius/Fahrenheit toggle
Mobile-responsive design
Animated weather icons
Clean, modern UI with custom styling

Demonstrates React component architecture, API integration, state management, and responsive web design principles.
